pyiron_contrib - Additional in-development modules for `pyiron <https://pyiron.readthedocs.io/en/latest/>`_.

pyiron_contrib  is developed in the `Computational Materials Design department <https://www.mpie.de/CM>`_ of
`Joerg Neugebauer <https://www.mpie.de/person/43010/2763386>`_ at the `Max Planck Institut f√ºr Eisenforschung (Max
Planck Institute for iron research) <https://www.mpie.de/2281/en>`_.

.. note::

    **pyiron_contrib**: This is the documentation page for experimental modules of pyiron.  If you're new to
    pyiron and want to get an overview head over to `pyiron <https://pyiron.readthedocs.io/en/latest/>`_.  If you're
    looking for the API docs of the atomistic simulation packages check `pyiron_atomistics
    <https://pyiron_atomistics.readthedocs.io/en/latest/>`_.
